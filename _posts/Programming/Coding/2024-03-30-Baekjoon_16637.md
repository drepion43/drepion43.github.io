## 괄호 추가하기(16637번)

<https://www.acmicpc.net/problem/16637>

### 문제
길이가 N인 수식이 있다. 수식은 0보다 크거나 같고, 9보다 작거나 같은 정수와 연산자(+, -, ×)로 이루어져 있다. 연산자 우선순위는 모두 동일하기 때문에, 수식을 계산할 때는 왼쪽에서부터 순서대로 계산해야 한다. 예를 들어, 3+8×7-9×2의 결과는 136이다.

수식에 괄호를 추가하면, 괄호 안에 들어있는 식은 먼저 계산해야 한다. 단, 괄호 안에는 연산자가 하나만 들어 있어야 한다. 예를 들어, 3+8×7-9×2에 괄호를 3+(8×7)-(9×2)와 같이 추가했으면, 식의 결과는 41이 된다. 하지만, 중첩된 괄호는 사용할 수 없다. 즉, 3+((8×7)-9)×2, 3+((8×7)-(9×2))은 모두 괄호 안에 괄호가 있기 때문에, 올바른 식이 아니다.

수식이 주어졌을 때, 괄호를 적절히 추가해 만들 수 있는 식의 결과의 최댓값을 구하는 프로그램을 작성하시오. 추가하는 괄호 개수의 제한은 없으며, 추가하지 않아도 된다.


### 입력

첫째 줄에 수식의 길이 N(1 ≤ N ≤ 19)가 주어진다. 둘째 줄에는 수식이 주어진다. 수식에 포함된 정수는 모두 0보다 크거나 같고, 9보다 작거나 같다. 문자열은 정수로 시작하고, 연산자와 정수가 번갈아가면서 나온다. 연산자는 +, -, * 중 하나이다. 여기서 *는 곱하기 연산을 나타내는 × 연산이다. 항상 올바른 수식만 주어지기 때문에, N은 홀수이다.

### 해결법
우선 괄호를 적절히 넣어 최대값을 만들어라는 문제이니, 재귀법인 DFS로 해결하기 위해 접근했습니다.   
예를 들어 3 + 8 * 7 - 9 * 2의 수식이 들어왔을 때, 괄호를 적절히 넣어 최대값을 도출하는데, 괄호의 경우 괄호 사이에는 한개의 연산 기호만 포함될 수 있다는 점을 생각해야합니다. 즉, (8 * 7)=56이라는 값을 바로 계산한 후 식에 대입해주면, 3 + 56 - 9 * 2 의 수식이 되고 문제의 조건처럼 더 이상 괄호를 넣지 않는다면 이 수식을 왼쪽에서부터 계산하면 됩니다. 계산하는 기능은 파이썬에서 제공하는 eval함수를 이용하여 string을 넣어 계산했습니다. 우선 가장 왼쪽인 3 + 8에는 괄호를 넣나 안넣나 동일한 결과가 나오기 때문에 그 이 후인 8 * 7 - 9 * 2에서 괄호를 어떻게 추가하면되는지에 대해 고민하면 됩니다. 만약 처음인 3 + 의 수식이 나왔는데, 그 뒤에 괄호가 포함된 경우이면 숫자 3을 DFS에 input을 넣었다면, 0번째 index이기 때문에 뒤의 연산은 2~4까지의 index를 괄호로 포함됬다 가정하고 계산하면 됩니다. 즉, 현재 들어온 수의 index에서 index+2~index+4까지를 고려해주면 됩니다. 따라서 index+4했을 때가 수식의 길이보다 작다면 괄호를 넣어보고, 또 괄호 안넣는 연산인 index+2보다 작다면괄호를 넣지 않는 3 + 8의 계산을 구해서 추가해주면 됩니다.

```python
n = int(input())
calc = input()
result = -float(1e9)

# 괄호 넣기
def dfs(depth, val):
    global result
    if depth == n:
        # do something
        result = max(result, int(val))
        return
    # 괄호 연산
    if depth + 4 <= n:
        # 3 + 8 * 7 - 9 * 2
        # 한개의 연산기호 나온 이후에 뒤쪽이 괄호가 있는 경우 : e.g. 3 +
        f_val = val + calc[depth]
        # 뒤 쪽에 괄호가 되어 있어 먼저 연산하는 경우 : e.g. (8 * 7)
        b_val = eval(calc[depth+1:depth+4])
        tmp = str(eval(f_val + str(b_val)))
        dfs(depth + 4,tmp)
    # 괄호 미포함 연산(왼쪽부터 차례대로 연산)
    if depth + 2 <= n:
        # 3 + 8 * 7 - 9 * 2
        # 3 + 8, 11*7, 77 -9 ...
        f_val = val + calc[depth:depth+2]
        tmp = str(eval(''.join(f_val)))
        dfs(depth+2, tmp)
dfs(1, calc[0])
print(result)
```
